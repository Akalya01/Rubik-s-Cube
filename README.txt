# 🧊 Rubik’s Cube Solver – Step-by-Step

A beginner-friendly interactive Rubik’s Cube simulator and step-by-step solver using **HTML, CSS, and JavaScript (OOP)**.

This app allows you to:
- Rotate individual faces of a cube
- Scramble the cube randomly
- Solve the cube **step-by-step** by clicking “Next Step”

---

## 🚀 Features

✅ Object-oriented cube representation  
✅ Manual rotation for all 6 faces (L, R, U, D, F, B)  
✅ Scramble button (generates a random state)  
✅ Step-by-step solution with **Next Step** button  
✅ Clean professional UI with **animated output**  
✅ Fully responsive layout and mobile-friendly  

---

## 🧩 Technologies Used

- **HTML5** – Structure  
- **CSS3** – Styling (Glassmorphism & Professional Theme)  
- **JavaScript (ES6)** – Cube logic & interactivity
